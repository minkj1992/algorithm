## 1st try
- bfs 사용
- 우선순위 개념을 도입함
  - `deque`를 사용한다
  - `*2`는 appendleft하여 우선순위를 최선으로 만들어준다.
  - `cur = Q.popleft()`
  
```python
from collections import deque
def nextPos(cur,nex,flag):
    if (0<= nex<= MAX) and (0==visit[nex]):
        visit[nex]=visit[cur]
        if not flag: 
            visit[nex]+=1
            Q.append(nex)
        else:
            Q.appendleft(nex)
N,K = map(int,input().split())
if N>=K:print(N-K)
else:
    MAX=10**5
    visit=[0]*(MAX+1)
    Q = deque([N])
    while Q:
        cur = Q.popleft()
        if cur == K: break
        nextPos(cur,cur*2,True)
        for i in (cur+1,cur-1):
            nextPos(cur,i,False)
    print(visit[K])
```
