```sql
SET @hour = -1;
SELECT
    @hour := @hour+1 AS HOUR,
    (SELECT COUNT(*) FROM ANIMAL_OUTS WHERE HOUR(DATETIME) = @hour) AS COUNT
FROM
    ANIMAL_OUTS
WHERE
    @hour < 23
```

- `:=` 은 오른쪽을 왼쪽으로 대입한다는 뜻
- `where`절을 빼니까 0..100 까지의 hour count를 보여주었다.
